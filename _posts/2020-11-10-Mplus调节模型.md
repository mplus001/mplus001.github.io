## 前言
之前整理过Mplus的中介模型语句，这一次打算将调节语句整理一遍。

### 1.简单调节模型（二分观察）
语句解释：
```
TITLE:调节变量为二分观察变量

DATA:
  FILE IS ...;

VARIABLE:
  NAMES ARE ...;
  
  USEVARIABLES ARE ...;

  Grouping is 调节变量(0=LOW,1=HIGH);

ANALYSIS:
  ESTIMATOR IS ML;
  BOOTSTRAP = 1000;

MODEL:
    !MEASUREMENT MODEL
    X BY x1 x2 x3;
    Y BY y1 y2 y3;

    !STRUCFTUAL MODEL
    Y ON X(a1); !LOW组的设定

    MODEL HIGH:!HIGH组的设定
    Y ON X(a2);
    
    MODEL CONSTRAINT:
    NEW(XY1, XY2, DIFF);
    XY1 = a1;
    XY2 = a2;
    DIFF = UFBI1 - UFBI2;

OUTPUT: SAMPSTAT CINTERVAL(BOOTSTRAP);
```
### 2.多条调节模型（二分观察）
语句解释：
```
TITLE:调节变量为二分观察变量时的多个调节路径

DATA:
  FILE IS ...;

VARIABLE:
  NAMES ARE ...;

  USEVARIABLES ARE ...;

  GROUPING IS 调节变量(0=LOW,1=HIGH);

ANALYSIS:
  ESTIMATOR IS ML;
  BOOTSTRAP = 1000;
MODEL:
  !MEASUREMENT MODE
    X1 BY x11 x12 x13;
    X2 BY x21 x22 x23;
    X3 BY x31 x32 x33;
    Y BY y1 y2 y3;


  !STRUCTUAL MODEL
  Y ON X1(a1); 
  Y ON X2(b1);
  Y ON X3(c1); 

  MODEL HIGH:!设置HIGH组
  Y ON X1(a2); 
  Y ON X2(b2);
  Y ON X3(c2);
  
  MODEL CONSTRAINT:
  NEW(X1Y-1,X2Y-2,X2Y-1,X2Y-2,X3Y-1,X3Y-2,DIFF1,DIFF2,DIFF3);
  X1Y-1 = a1; 
  X1Y-2 = a2;
  X2Y-1 = b1;
  X2Y-2 = b2;
  X3Y-1 = c1;
  X3Y-2 = c2;
  DIFF1 = a1 - a2;
  DIFF2 = b1 - b2;
  DIFF3 = c1 - c2;

OUTPUT: SAMPSTAT CINTERVAL(BCBOOTSTRAP);
```
### 3.简单调节模型（连续变量）
语句解释：
```
TITLE:调节变量为潜在变量

DATA:
  FILE IS TAM.dat;

VARIABLE:
  NAMES ARE...;
  
  USEVARIABLES ARE ...;

ANALYSIS:
  TYPE IS GENERAL RANDOM;!写MLR必写的一句话，因为交互作用，所以效果是随机的
  ESTIMATOR IS MLR;      !因为要做交互作用，所以要用MLR估计
  ALGORITHM IS INTEGRATION;
  CONVERGENCE IS 0.01; !收敛标准

MODEL:
  ! MEASUREMENT MODEL
  X BY x1 x2 x3;
  Y BY y1 y2 y3;
  Mo BY mo1 mo2 mo3;

  ! STRUCTURAL MODEL 
  XMo | X XWITH Mo; !给自变量与调节变量的交互项命名
  Y ON X Mo XMo;

OUTPUT:SAMPSTAT;
```
### 4.两个调节变量（连续）的模型
```
TITLE:调节变量为潜在变量且有两个(再练习)

DATA:
  FILE IS TAM.dat;

VARIABLE:
  NAMES ARE...;
  
  USEVARIABLES ARE ...;

ANALYSIS:
  TYPE IS GENERAL RANDOM;!写MLR必写的一句话，因为交互作用，所以效果是随机的
  ESTIMATOR IS MLR;
  ALGORITHM IS INTEGRATION;
  CONVERGENCE IS 0.01; !收敛标准

MODEL:
  ! MEASUREMENT MODEL
  X BY x1 x2 x3;
  Y BY y1 y2 y3;
  Mo1 BY mo11 mo12 mo13;
  Mo2 BY mo21 mo22 mo23;
  
  ! STRUCTURAL MODEL 
  XMo1 | X XWITH Mo1; !给自变量与调节变量的交互项命名
  XMo2 | X XWITH Mo2;
  Y ON X Mo1 Mo2 XMo1 XMo2;

OUTPUT:SAMPSTAT;
```
